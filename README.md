# qooxdoo/demobrowser
A GUI framework application for qooxdoo demos

## Developing DemoBrowser
Clone this repo and compile it:

```
    $ git clone https://github.com/qooxdoo/qxl.demobrowser
    $ cd qxl.demobrowser
    $ qx contrib install
    $ npm install
    $ qx serve
```
Then open [http://localhost:8080](http://localhost:8080)
